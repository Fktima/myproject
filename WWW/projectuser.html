This is easy to use this website

This website gives you information about Chromosome 21. All you need is putting your entry in the box and select the related option.

After you hit the submit button, this website will show you:

  • Access to search the database based on your entry.
  • Codon usage frequencies within the coding region with the overall codon usage information for all sequences.
  • The complete DNA sequence with the coding regions highlighted
  • Identifying sticky-end restriction enzyme sites in the genomic DNA.
  • List of all enteries you can search on them.
